# resume_download 函数

| 项目 | 说明 |
|------|------|
| **描述** | 恢复已暂停的下载。核心版本 ≥0.5.1 后无法恢复已停止的下载。 |
| **参数** | ***id*** - 下载器实例ID |
| **返回值** | 0（成功）或 -1（下载器不存在） |

## 版本差异

### 核心版本 0.5.0
- **无法恢复**下载
- 因为 ***pause_download*** 会立即从 map 中删除下载器
- ***resume_download*** 会找不到下载器而返回 -1

### 核心版本 0.5.1
- **可以恢复**已暂停的下载
- 因为 ***pause_download*** 保留了下载器在映射表中
- ***resume_download*** 内部调用 ***resume_download()*** 方法：
  1. 等待所有旧任务退出（***activeTasks.Wait()***）
  2. 重新创建 context 并调用 ***start_download()*** 重新开始下载
- **注意**：无法恢复已停止（***stop_download***）的下载，因为下载器已被销毁
