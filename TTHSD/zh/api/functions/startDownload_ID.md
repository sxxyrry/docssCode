# startDownload_ID 函数

| 项目 | 说明 |
|------|------|
| **描述** | 启动已创建的下载器（顺序下载）。 |
| **参数** | ***id*** - 下载器实例ID |
| **返回值** | 0（成功）或 -1（下载器不存在） |

## 版本差异

### 核心版本 0.5.0
- 启动失败时发送 ***EventTypeMsg*** 类型的错误事件
- 错误信息在 ***msg["Text"]*** 中

### 核心版本 0.5.1
- 启动失败时发送 ***EventTypeErr*** 类型的错误事件
- 错误信息在 ***msg["Error"]*** 中
- 如果错误是 ***context.Canceled***，则不发送错误事件

---

> **注意**：如果您使用的是 TTHSD Core 0.5.0 版本，启动失败时请监听 ***EventTypeMsg*** 事件并从 ***msg["Text"]*** 中获取错误信息。
